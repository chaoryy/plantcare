# PlantCare AI

## Project Overview
PlantCare AI is a smart assistant for plant care and home maintenance.  
The system identifies plants using AI, analyzes their condition from images, and creates personalized care schedules based on weather data.

---

## Repository Structure

### Frontend
User interface of the application (web app).

Includes:
- Dashboard
- Plant upload page
- Plant cards
- Results / analysis page

---

### Backend
Server-side logic and API.

Includes:
- User management
- Plant database
- API endpoints
- Business logic for scheduling care reminders

---

### AI Module
AI-based image analysis system.

Includes:
- Plant recognition from images
- Health condition analysis
- AI prompts and response processing
- JSON output formatting

---

### DevSecOps / Infrastructure
Deployment and system configuration.

Includes:
- Docker setup
- Environment variables (.env)
- CI/CD pipelines
- Security configuration

---

### Documentation
Project documentation and team materials.

Includes:
- Product requirements
- Architecture notes
- API documentation
- Presentation materials
